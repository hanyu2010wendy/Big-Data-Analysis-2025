## Title
Build a WDI Panel Dataset in Python | Big Data Analysis with Python

## Short description
Learn how to clean World Bank WDI data, filter one country, reshape the dataset into panel format, and inspect missing values in Python.

## Long description
In this lesson, I show how to work with the World Development Indicators dataset in Python. We read the raw file, filter one country, reshape the data from wide format into panel format, export the cleaned dataset, and check missing values. This workflow is useful for classroom learning, research preparation, and GitHub portfolio building.
