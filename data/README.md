# Data Folder Notes

## `wdi/`
- `WDI_course_subset.csv`: WDI teaching subset used by the notebooks.
- `course_country_list.csv`: list of countries kept in the subset.
- `country_assignment_template.csv`: placeholder template added because the original assigned-country file was missing.

## `china/`
- `china_panel_data.xlsx`: province-level China panel data used in modules 2, 4, and 7.
- `china_corporate_units_by_region_2020.xlsx`: supplementary NBS dataset used in module 2.
- `china_administrative_division_codebook.xlsx`: administrative-division codebook.
- `china_panel_data_dictionary.csv`: English aliases and explanations for the main panel-data variables.
- `province_name_mapping.csv`: Chinese-to-English province mapping.
- `guangxi.xls`, `hebei.xls`, `jilin.xls`: province data files used in Homework 2.

## `maps/`
- `china_provinces.*`: shapefile components used in the geographic mapping lecture.
